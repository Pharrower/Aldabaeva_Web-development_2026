# Changelog

Все важные изменения в проекте автоматизации волонтерского приюта «Доброе сердце» будут фиксироваться в данном файле.

Проект следует принципам семантического версионирования (Semantic Versioning).

## [1.0.0] — 2026-05-01
### Added
- Интегрирована двухконтурная система валидации входящих данных (клиентская на JavaScript + серверная на базе аннотаций Jakarta Bean Validation).
- Реализован асинхронный механизм отправки веб-форм AJAX через Fetch API в каталоге животных без перезагрузки страниц.
- Подготовлен скрипт инициализации структуры реляционной базы данных `db/init.sql` для СУБД PostgreSQL 17.
- Оформлена финальная редакция пояснительной записки в каталоге `docs/`.

## [0.2.0] — 2026-04-20
### Added
- Разработан динамический пользовательский интерфейс веб-приложения на базе шаблонизатора Thymeleaf и CSS-фреймворка Bootstrap 5.
- Добавлен интерактивный компонент прогресс-бара (Progress Bar) для отображения уровня заполненности склада материального обеспечения приюта.
- Спроектирован и настроен конфигурационный контур веб-сервера Apache Tomcat 10.1 (перевод сетевой инфраструктуры на выделенный порт `8082`).

## [0.1.0] — 2026-03-15
### Added
- Инициализирован базовый каркас приложения на Spring Boot 3.x с использованием Java 25.
- Настроена доменная модель предметной области (пакет `entities`) согласно паттерну PCMEF (`User`, `Animal`, `AdoptionRequest`, `Supply`).
- Реализован слой доступа к данным (пакет `repositories`) на базе Spring Data JPA.
- Интегрирован пул соединений HikariCP для связи с PostgreSQL.
