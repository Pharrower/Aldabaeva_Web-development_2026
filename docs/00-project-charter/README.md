# Информационная система для приюта "Доброе сердце"

## 1. Цель проекта
Разработка веб-ориентированной системы для автоматизации деятельности волонтёрского приюта, направленной на повышение эффективности работы с карточками животных и учета материальной помощи.

## 2. Основные задачи
- Автоматизация процесса учета животных и их статусов.
- Организация системы подачи заявок на адопцию (усыновление).
- Ведение учета складских запасов (корма, медикаменты).
- Интеграция с системой регистрации волонтеров и донатов.

## 3. Стек технологий
- **Backend:** Java 21, Spring Boot 3.
- **Frontend:** HTML5, Thymeleaf, Bootstrap.
- **Database:** PostgreSQL.
- **Server:** Apache Tomcat (развертывание через .war).
