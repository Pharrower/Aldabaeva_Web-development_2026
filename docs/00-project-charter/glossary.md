# Глоссарий предметной области (Glossary)

* **Адопция (Усыновление)** — юридический и фактический процесс передачи домашнего животного из приюта на постоянное содержание новому владельцу (собственнику).
* **Волонтер (Администратор приюта)** — авторизованный пользователь информационной системы, обладающий правами на редактирование каталогов, изменение статусов заявок и ведение складского учета.
* **Заявка на адопцию (Adoption Request)** — документ (электронная форма), формируемый потенциальным владельцем, содержащий контактные данные и текстовое обоснование намерения забрать животное.
* **Материальное обеспечение (Supply)** — совокупность материальных ресурсов (корма, лекарственные препараты, инвентарь), необходимых для поддержания жизнедеятельности приюта.
* **Карточка питомца (Animal Entity)** — структурированная совокупность данных о конкретном животном в базе данных, включающая кличку, биологический вид, возраст, описание характера и текущий статус нахождения в приюте.
