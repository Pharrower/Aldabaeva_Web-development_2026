# Паспорт проекта: Информационная система приюта «Доброе сердце»

## Общие сведения
* **Название проекта:** Система автоматизации учета и управления приютом для животных.
* **Цель:** Оптимизация бизнес-процессов приюта, повышение эффективности учета ресурсов и ускорение процесса адопции (пристройства) питомцев.
* **Заказчик:** Администрация приюта «Доброе сердце».

## Ключевые этапы (Milestones)
1. Проектирование архитектуры и бизнес-логики.
2. Реализация базы данных и серверной части (REST API).
3. Разработка адаптивного пользовательского интерфейса (UI).
4. Тестирование и развертывание на сервере Apache Tomcat.

## Основные метрики успеха
* Снижение времени на заполнение отчетности по складу на 40%.
* Сокращение времени обработки заявки на адопцию с 3 дней до 1 часа.
