# Этап 1: Бизнес-моделирование предметной области (Business Modeling)

## Описание этапа
На данном этапе произведено комплексное предпроектное обследование волонтерского приюта для бездомных животных «Доброе сердце». Целью этапа является формализация существующих бизнес-процессов организации, выявление ключевых проблем ручного учета ресурсов и обоснование необходимости разработки веб-ориентированной информационной системы.

## Содержимое каталога
* [glossary.md](glossary.md) — Глоссарий (терминологический словарь) предметной области.
* [context-diagram.md](context-diagram.md) — Контекстная диаграмма взаимодействия системы с внешней средой (IDEF0).
* [buc-diagram.md](buc-diagram.md) — Диаграмма бизнес-прецедентов (Business Use Case Diagram).

## Результаты анализа
В ходе бизнес-анализа (на основе SWOT-матрицы проекта) было определено, что автоматизация учета карточек питомцев, склада материального обеспечения и входящих заявок на усыновление позволит снизить операционные издержки волонтеров на 40% и исключить риск потери критически важных данных.
