# Этап 6: Реализация системы и обеспечение качества (Implementation & QA)

## Описание этапа
На данном этапе осуществлена непосредственная программная реализация (кодирование) проектных решений на платформе **Java 25** с использованием фреймворка **Spring Boot 3.x**. Особое внимание уделено декомпозиции исходного кода по пакетам в соответствии со слоями архитектурного шаблона PCMEF и покрытию критически важной бизнес-логики модульными тестами (Unit Testing) на базе JUnit 5.

## Содержимое каталога
* [code-structure.md](code-structure.md) — Подробное описание структуры исходного кода, классов и логических интерфейсов проектирования.
* [tests.md](tests.md) — Документирование процесса автоматизированного тестирования (результаты прохождения тест-кейсов).

## Спецификация технологического стека реализации
* **Язык программирования:** Java JDK 25 (Eclipse Adoptium)
* **Ядро бэкенд-платформы:** Spring Framework 6 / Spring Boot 3
* **Объектно-реляционное отображение (ORM):** Hibernate 6 / Spring Data JPA
* **Среда модульного тестирования:** JUnit Jupiter 5.10 + Mockito
