# Тестирование проекта (Testing & QA)

В данном разделе представлена документация и конфигурация системы тестирования, обеспечивающей надежность и корректность работы приюта.

## Стратегия тестирования
Система использует многоуровневый подход к обеспечению качества (Testing Pyramid):
1. **Unit-тестирование:** Проверка изолированной логики (Entities, DTO, вспомогательные методы) с помощью JUnit 5.
2. **Интеграционное тестирование:** Проверка взаимодействия сервисов с контекстом Spring.
3. **API-тестирование:** Валидация эндпоинтов контроллеров с использованием `MockMvc`.

## Отчетность и метрики
Для контроля покрытия кода используется плагин **JaCoCo**.
- **Текущий уровень покрытия (Line Coverage):** 49%.
- **Критически важные узлы (DTO, Entities):** 96–100%.

## Инструкция по запуску
Для запуска всех тестов и генерации свежего отчета JaCoCo выполните в терминале команду:

    ```bash
    mvn clean verify
    ```
После выполнения команды отчет будет доступен по пути:
    target/site/jacoco/index.html
