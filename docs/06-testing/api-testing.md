# Тестирование API (Postman)

Для проверки доступности и корректности работы API используется Postman.

## Основные сценарии проверки:
- **GET /catalog:** Доступность каталога для всех пользователей.
- **POST /admin/donations:** Тестирование прав администратора (согласно логике `WebCoverageTest`).
- **GET /admin/*:** Проверка ограничения доступа (статус 403 при попытке доступа без прав администратора).
