# План тестирования (Test Plan)

## Стратегия
Система приюта использует многоуровневый подход к обеспечению качества:
1. **Unit-тестирование:** Изолированная проверка логики (сущности, DTO, расчеты).
2. **Интеграционное тестирование:** Проверка взаимодействия сервисов с контекстом Spring.
3. **API-тестирование:** Функциональная проверка эндпоинтов.

## Инструментарий
- **JUnit 5 / Mockito:** Основа для написания unit и интеграционных тестов.
- **JaCoCo:** Инструмент контроля покрытия кода (текущий результат: 42% Line Coverage).
- **Postman:** Используется для ручного тестирования API и проверки безопасности ролей (ADMIN/USER).
