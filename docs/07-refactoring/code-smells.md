# Устраненные запахи кода

| Запах               | Что сделали                                                                                  |
| :------------------ | :------------------------------------------------------------------------------------------- |
| **God Object**      | Вынесли логику Supplies, Donations, Volunteers в отдельные сервисы.                          |
| **Feature Envy**    | Перенесли логику обработки данных из контроллеров внутрь сервисов.                           |
| **Hardcoded logic** | Заменили прямое обращение к репозиториям в контроллерах на вызовы сервисов (Facade pattern). |
| **Tight Coupling**  | Заменили полевую инъекцию на конструкторную, что развязало зависимости.                      |
