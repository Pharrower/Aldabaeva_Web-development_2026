# Техническое задание на разработку ИС приюта «Доброе сердце»

## 1. Общие сведения
* **Наименование системы:** Информационная система автоматизации волонтерского приюта для бездомных животных «Доброе сердце».
* **Основание для разработки:** Курсовой проект по дисциплине «Программная инженерия».
* **Разработчик:** студентка 3 курса Алдабаева Виктория.

## 2. Назначение и цели создания системы
* **Назначение:** Автоматизация операционных процессов учета фонда животных, координации волонтерской деятельности, ведения складского учета материально-технического обеспечения и обработки входящих заявок граждан.
* **Цели:** Снижение временных затрат волонтеров на рутинный учет, исключение дублирования данных, повышение прозрачности деятельности приюта для спонсоров и повышение конверсии успешных передач животных в семьи (адопции).

## 3. Требования к системе
### 3.1. Требования к функциональным подсистемам:
1. **Подсистема каталога:** Отображение списка животных, фильтрация по видам, статусам, возрасту.
2. **Подсистема адопции:** Прием заявок через веб-форму, отправка данных на сервер в асинхронном режиме без перезагрузки (AJAX), автоматическая валидация полей.
3. **Подсистема склада:** Мониторинг запасов снабжения (корма, медикаменты), расчет уровня заполненности в процентах, визуальное отображение шкал (Progress Bars).

### 3.2. Требования к техническому обеспечению:
* Сервер приложений с поддержкой спецификаций сервлетов Jakarta EE 10 (Apache Tomcat 10.1).
* Реляционная СУБД PostgreSQL версии 17 или выше.
* Поддержка браузеров: Google Chrome, Mozilla Firefox, Яндекс.Браузер, Apple Safari актуальных версий.
