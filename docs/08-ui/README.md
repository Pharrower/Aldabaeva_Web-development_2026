# Проектирование и разработка пользовательского интерфейса (User Interface Design)

## Описание этапа
На данном этапе спроектирован и реализован адаптивный графический интерфейс пользователя (UI) веб-приложения приюта «Доброе сердце». Слой представления (Presentation Layer в рамках паттерна PCMEF) построен на базе современных веб-стандартов HTML5 и CSS3 с использованием шаблонизатора **Thymeleaf**, стилизации фреймворком **Bootstrap 5** и клиентских скриптов на **JavaScript (Fetch API)**.

## Принципы проектирования UI
1. **Адаптивность (Responsive Design):** Благодаря сеточной системе Bootstrap 5 (`container`, `row`, `col-md-*`), интерфейс корректно масштабируется под любые типы устройств — от мобильных телефонов волонтеров до стационарных мониторов в административном блоке приюта.
2. **Интуитивность и доступность:** Навигация организована через сквозное верхнее меню (Navbar). Важные показатели (заполненность склада) визуализированы графически, а формы обратной связи снабжены понятными подсказками.
3. **Компонентный подход:** Интерфейс разбит на переиспользуемые блоки (карточки животных, таблицы ресурсов, модальные окна), что упрощает его расширение.
4. **Обратная связь (Feedback Loops):** Взаимодействие с пользователем сопровождается визуальными уведомлениями (Flash-сообщения при успешной подаче заявки, валидация полей форм в реальном времени), что минимизирует вероятность ошибок при вводе данных.
