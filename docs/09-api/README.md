# Документирование API (OpenAPI & Swagger)

В данном разделе представлена спецификация RESTful API проекта "Приют". Документация реализована с использованием стандарта OpenAPI 3.0 и автоматизирована через Swagger UI.

## 🛠 Технологический стек
* **Библиотека:** `springdoc-openapi-starter-webmvc-ui`
* **Стандарт:** OpenAPI 3.0.3

## 🎯 Задачи этапа
1. Автоматизация генерации документации на основе исходного кода.
2. Обеспечение удобного интерфейса для тестирования эндпоинтов (Swagger UI).
3. Соблюдение контракта взаимодействия между клиентом и сервером.

- [Общая спецификация и стандарты](api-documentation.md) — концепция и описание методов.
- [Примеры контрактов (JSON)](api-examples.md) — наглядные примеры запросов и ответов.
