# Развертывание (Deployment)

В данном разделе описан процесс ручного развертывания веб-приложения на сервере Apache Tomcat.

## 🛠 Техническое окружение
* **Сервер:** Apache Tomcat 10.x
* **СУБД:** MySQL/PostgreSQL
* **Формат:** WAR-архив

## 📋 Инструкция по деплою
1. **Подготовка:** Убедитесь, что в `pom.xml` указан `packaging: war`.
2. **Сборка:** Выполните команду `mvn clean package`.
3. **Развертывание:** Переместите файл `shelter.war` из папки `target/` в директорию `webapps/` вашего Tomcat.
4. **Запуск:** Перезапустите сервер или дождитесь, пока Tomcat автоматически распакует архив.
