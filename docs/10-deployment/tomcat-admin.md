# Администрирование Apache Tomcat

Для обеспечения стабильной работы приложения на сервере Tomcat используются следующие настройки:

## 1. Логирование

Все логи приложения доступны в директории `logs/`. 
* `catalina.out` — основной лог запуска и работы сервера.
* `localhost.[дата].log` — логи приложения (ошибки Spring Boot).

## 2. Настройка памяти (JVM)

Если приложению не хватает ресурсов, параметры памяти задаются в файле `bin/setenv.sh`:
```bash
export CATALINA_OPTS="-Xms512m -Xmx1024m"
```

## 3. Подключение к БД

Приложение подключается к базе данных через параметры в application.properties. В продакшн-окружении рекомендуется использовать JNDI-источник данных в context.xml сервера для обеспечения безопасности паролей.
