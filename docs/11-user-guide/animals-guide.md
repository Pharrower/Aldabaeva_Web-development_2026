# Работа с каталогом питомцев

Этот раздел позволяет управлять карточками животных.

## Как добавить нового питомца?
1. Перейдите на вкладку «Животные».
2. Нажмите кнопку **«Добавить животное»**.
3. Заполните форму: кличка, возраст, вид.
4. Загрузите фотографию (рекомендуемый формат: JPG/PNG).
5. Нажмите **«Сохранить»**.

## Статусы животных
* **Свободен:** Животное готово к передаче в новую семью.
* **На испытательном сроке:** Животное уже нашло временную семью, но юридически еще закреплено за приютом.
* **Пристроен:** Животное успешно нашло постоянный дом.
