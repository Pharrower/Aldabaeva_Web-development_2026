# Пожертвования и финансовая помощь

Этот раздел помогает сотрудникам приюта отслеживать материальную поддержку со стороны благотворителей.

## Просмотр списка донатов
1. Перейдите в раздел **«Администрирование донатов»**.
2. В таблице отображается весь список транзакций: имя донора, сумма и дата пожертвования.

## Статистика
В верхней части страницы автоматически рассчитывается **«Общая сумма пожертвований»**. Это значение обновляется после каждой новой записи, что позволяет быстро подводить итоги отчетного периода без ручных вычислений.

## Назначение раздела
Данный функционал используется для:
* Формирования финансовой отчетности приюта.
* Публичной демонстрации прозрачности доходов (для страницы «Помощь»).
