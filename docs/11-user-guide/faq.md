# Часто задаваемые вопросы (FAQ)

**Вопрос:** Почему я не вижу новую заявку от пользователя?
**Ответ:** Проверьте вкладку «Заявки» в административной панели. Если заявки нет, попробуйте обновить страницу (клавиша F5).

**Вопрос:** Как изменить роль волонтера?
**Ответ:** Изменение ролей доступно только пользователям с правами `ADMIN` в разделе «Управление пользователями».

**Вопрос:** Где посмотреть, сколько осталось корма?
**Ответ:** Данные по складу обновляются в реальном времени на главной странице дашборда.
