# Управление складом снабжения

Модуль предназначен для контроля наличия необходимых ресурсов для животных.

## Просмотр состояния склада
На главной странице администратора расположен интерактивный индикатор (Progress Bar):
* **Зеленый цвет:** Запасы в норме.
* **Красный цвет:** Критический дефицит (требуется немедленное пополнение).

## Добавление новой поставки
1. Перейдите в раздел **«Администрирование склада»**.
2. В поле «Новая поставка» введите название ресурса (например, «Корм для собак»).
3. Укажите количество.
4. Нажмите **«Сохранить»**.

## Удаление записи
Если ресурс был полностью израсходован или данные были внесены ошибочно, используйте кнопку **«Удалить»** напротив соответствующей записи в таблице.
