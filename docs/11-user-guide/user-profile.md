# Работа с профилем пользователя

Раздел предназначен для настройки учетных записей волонтеров и администраторов.

## Вход в систему
1. Перейдите на страницу входа.
2. Введите ваш логин и пароль.
3. Нажмите кнопку **«Войти»**.

## Личный кабинет
В личном кабинете вы можете:
* **Просмотреть свои данные:** ФИО, контактную информацию и присвоенную роль (`ADMIN` или `VOLUNTEER`).
* **Моя активность:** Таблица с последними выполненными вами действиями (добавление животных, обновление остатков на складе), что обеспечивает прозрачность работы сотрудников.

## Безопасность
* При завершении работы на общедоступном компьютере обязательно нажмите кнопку **«Выход» (Logout)**.
* Не передавайте свои учетные данные третьим лицам.
